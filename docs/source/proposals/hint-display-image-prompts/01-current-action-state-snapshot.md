# Current Action / State Snapshot Prompt

## Goal

Visualize the lightest default hint: a one-glance recovery aid that answers "What is happening right now?"

This should feel like a tiny scene-state checkpoint, not a summary card.

## Image Prompt

Create a high-fidelity UI mockup of a calm long-form fiction reader. Show a page of novel text on the left. Near the start of a new scene, show one subtle inline state hint just above or beside the paragraph, no more than one visible hint.

The hint should look like a compact "Now" snapshot with 3 short rows:

- place
- active cast
- immediate goal or problem

Use a small slate/zinc accent. The reader should still visually focus on the prose. The support surface should be minimal, like a quiet inline strip or a slim right-side panel that is already open because the scene just changed.

The design should communicate:

- this is the one hint allowed to be visible by default
- it helps re-enter the current scene state
- it does not replace reading the text
- it is anchored to the current scene boundary

Use placeholder grey lines for the novel text. Use only short readable labels like "Now", "Place", "Cast", "Problem". Keep the layout clean, quiet, and sparse.

## Negative Prompt

No large summary card, no paragraph-length explanation, no colorful dashboard, no timeline, no infographic, no modal, no decorative hero layout.

## Success Criteria

- The prose remains dominant.
- The hint is visibly useful but not intrusive.
- There is only one default-visible support element.
