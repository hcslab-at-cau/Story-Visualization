# Relation / Social Delta Prompt

## Goal

Visualize a hint that answers "What changed between these characters?"

This should show a before-now social relation update.

## Image Prompt

Create a high-fidelity UI mockup of the Story-Visualization reader. The left column shows a dialogue-heavy paragraph. Two character names are subtly marked with restrained relation-colored underlines. A small anchor popover offers "Relation".

On the right side, show a compact relation delta panel. The panel should have a three-part layout:

- "Before"
- "Now"
- "Why it matters"

Use a very restrained rose/mauve or neutral accent, not a romantic or dramatic style. The relation should look like a reading-comprehension aid, not a social network graph. A tiny connector line between two character chips is acceptable.

The design should communicate:

- relation support appears only when a meaningful shift exists
- it compares previous alignment with current alignment
- it is grounded in the current dialogue/action
- it stays optional unless the relation change is essential

Use short labels and grey placeholder lines.

## Negative Prompt

No large social graph, no dating-app style cards, no relationship drama poster, no dense network visualization, no permanent cast matrix.

## Success Criteria

- The before-now contrast is clear.
- The UI remains restrained and literary.
- The relation hint feels useful for following the scene, not decorative.
