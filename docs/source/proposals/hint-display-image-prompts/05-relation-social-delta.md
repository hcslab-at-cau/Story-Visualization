# Relation / Social Delta Prompt

## Goal

Visualize a hint that answers "What changed between these characters?"

This should show a before-now social relation update.

## Image Prompt

Create a high-fidelity UI mockup of the Story-Visualization reader. The reading column shows a dialogue-heavy paragraph. Two character names are subtly marked with restrained relation-colored underlines. A small anchor popover offers "Relation".

Show the relation hint as a slim side sheet or margin card only after the reader chooses "Relation". The first visible answer should be one short line, for example "Their relationship shifts from aligned to tense here." Use placeholder lines rather than exact prose.

Below that first answer, include a compact secondary before-now structure:

- "Before"
- "Now"
- a collapsed "why it matters" row

Use a very restrained rose/mauve or neutral accent, not a romantic or dramatic style. The relation should look like a reading-comprehension aid, not a social network graph. A tiny connector line between two character chips is acceptable, but keep it small.

The design should communicate:

- relation support appears only when a meaningful shift exists
- it gives the change in one sentence first, with previous/current detail secondary
- it is grounded in the current dialogue/action
- it stays optional unless the relation change is essential

Use short labels and grey placeholder lines.

## Negative Prompt

No large side panel, no large social graph, no dating-app style cards, no relationship drama poster, no dense network visualization, no permanent cast matrix.

## Success Criteria

- The before-now contrast is clear.
- The UI remains restrained and literary.
- The relation hint feels useful for following the scene, not decorative.
