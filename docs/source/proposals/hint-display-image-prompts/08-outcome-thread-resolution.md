# Outcome / Thread Resolution Prompt

## Goal

Visualize a spoiler-safe hint that answers "What earlier thread is being resolved now?"

This should show only already-revealed information and unresolved tension markers.

## Image Prompt

Create a high-fidelity UI mockup of a fiction reader. The left column shows a current scene paragraph. A subtle anchor near the paragraph says "Thread" or "Resolution".

On the right side, show a compact thread resolution panel. The panel should include:

- earlier thread
- now resolved or still open
- why this matters now

Use a restrained neutral plus soft amber accent. Include a tiny "spoiler safe" or "revealed so far" visual cue, but do not make it look like a warning. If an unresolved thread is shown, display it as an open question, not as a future prediction.

The design should communicate:

- the support never reveals future information
- it connects an earlier tension to the current scene
- it helps readers recognize closure or continuation
- it is optional and evidence-grounded

Use short labels and placeholder lines.

## Negative Prompt

No spoiler alert banner, no future prediction panel, no dramatic plot board, no full timeline of the novel, no mystery map.

## Success Criteria

- The design clearly distinguishes "resolved now" from "still open".
- It feels safe for the current reader position.
- It does not reveal or imply future plot.
